== Introduction

By harnessing the power of Red Hatâ€™s cloud partners, customers can benefit by extending their OpenShift Virtualization deployment in a truly hybrid environment. OpenShift Virtualization is supported across leading public cloud providers including: Amazon Web Services, Microsoft Azure, Google Cloud, IBM Cloud, and Oracle Cloud Infrastructure, giving customers flexibility to run their workloads where it makes the most sense for their organization. This module is primarily instructive, but includes both video demonstrations and interactive demos showcasing OpenShift Virtualization running across these cloud platforms. 
 Cloud availability plays a key role in modernization by enabling organization to migrate, modernize, and operate virtual machines alongside cloud-native applications on a single, consistent platform.By extending their virtualization workloads to the cloud provider of choice, customers can address many of the challenges that initially drive adoption, including the need for elastic scale, faster provisioning, and simplified operations through unified management and joint support. 

Running OpenShift Virtualization on cloud infrastructure also supports cost optimization strategies by allowing customers to right-size resources, take advantage of flexible consumption and committed spend models, and reduce the overhead associated with maintaining on-premises hardware.See the sections below to learn more about each of the cloud providers that we partner with, and how they support OpenShift Virtualization.


== Amazon Web Services

Red Hat OpenShift Virtualization is *Generally Available* in both self-managed Red Hat OpenShift, and fully-managed https://www.redhat.com/en/technologies/cloud-computing/openshift/aws[Red Hat OpenShift on AWS (ROSA)^] clusters in Amazon Web Services, simply by installing the Virtualization Operator, and adding bare metal compute nodes to your cluster.

== Conclusion

Having Red Hat OpenShift Virtualization available in our cloud partner environments allows for our customers to extend their datacenters into a fully hybrid scenario for both Virtual Machines and Containers. This enables resilient applications and redundant infrastructures across multiple regions and availability zones to ensure that services are always available. Red Hat OpenShift Cloud Services further modernizes operations by simplifying deployment and management, helping customers get up and running faster with less operational overhead.
