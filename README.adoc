= Accelerating Infrastructure Modernization with OpenShift Virtualization

Showroom lab guide for *LB2391*, a hands-on workshop covering VM foundations, Service Mesh integration, GitOps automation, Developer Hub self-service, and multicloud operations with Red Hat OpenShift Virtualization.

== Local Preview

[source,sh]
----
podman run --rm --name antora -v $PWD:/antora -p 8080:8080 -i -t ghcr.io/juliaaano/antora-viewer
----

For SELinux environments, append `:z` to the volume mount:

[source,sh]
----
podman run --rm --name antora -v $PWD:/antora:z -p 8080:8080 -i -t ghcr.io/juliaaano/antora-viewer
----

Then open http://localhost:8080.

== Modules

* *Module 01* -- VM foundations with OpenShift Lightspeed
* *Module 02* -- Service Mesh integration with virtual machines
* *Module 03* -- Declarative management of VMs with GitOps automation
* *Module 04* -- Self-service VM provisioning with Developer Hub
* *Module 05* -- OpenShift Virtualization in the cloud
